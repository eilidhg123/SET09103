To better understand how to build a HTML based web page, we first need to understand what each tag does and how they work

<html> </html>

We should always start with <html>

Addtionally we need to remember to close every tag we open, hence we will end our code for out webpage with </html> <head> </head>

The next thing we want to add is a head tag.

In this tag we put things that wouldn't be displayed within the actual web page, such as:

<link rel="stylesheet href="filename.css"> - this links a style sheet that allows us to customise the look of our web page

<title></title> - which allow us to add a title that will appear on the tab at the top of the browser

<body> </body> This indicates that this is the start of the code that will be displayed in our browser. The main body of our HTML will live here. <p> </p> This allows us to idicate different paragraphs of text, so that there can be a clear distiction between content when needed.
